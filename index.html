jQueryToast v0.2

Android-like Toast Plugin for jQuery

About

This is a simple plugin for displaying Android-like Toast messages. Call the plugin and it displays the message and fades away exactly as a toast on an android device would. Any messages that are created while another is being displayed will be put in a Que and displayed subsequently.

The plugin takes one parameter, the options object. This object passes the message and any other of the configurable options to customize the plugin. While you do not have to pass in the object at all, if you do not the toast message will be "This is a Toast".

Options

All parameters are passed to the plugin using an object. The options are as follows:

  • message <string> - Default: "This is a Toast"
  • displayTime <integer> - Default: 2000 - Time to display the Toast in milliseconds
  • inTime <integer> - Default: 300 - Duration of fadeIn in milliseconds
  • outTime <integer> - Default: 200 - Duration of fadeOut in milliseconds
  • maxWidth <integer> - Default: 400 - Maximum width in pixels
  • showDirect <boolean> - Default: false - If set, show the message immediately instead of queueing

Usage Example

<script>
  $(function(){
    $(document).click(function(){
      $.toast({
        message:"A Toast To You!",
        displayTime:4000,
        inTime:100,
        outTime:400
      });
    });
  });
</script>
